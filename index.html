¿Qué es la accesibilidad web?

La accesibilidad web tiene como objetivo lograr que las páginas web sean utilizables por el máximo número de personas, independientemente de sus conocimientos o capacidades personales, e independientemente de las características técnicas del equipo utilizado para acceder a la Web.

La necesidad de que la Web sea universal y accesible por cualquier persona está presente desde el principio de la Web, ya que era un requisito contemplado en su diseño por su creador Tim Berners-Lee. De hecho, la Web ofrece oportunidades sin precedentes a las personas con discapacidad, si bien, la falta de accesibilidad les creará graves barreras que impedirán su uso.


Objetivo de esta web

El objetivo de este proyecto es el desarrollo de un recurso web multi-página que ofrezca dos versiones distintas para cada una de ellas: la inaccesible y la accesible correspondiente, de acuerdo con las Pautas de Accesibilidad del Contenido Web del W3C.

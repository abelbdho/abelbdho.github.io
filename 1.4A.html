1. Perceptible - Primer Principio

La información y todo componente presentado en la interfaz de la web deben ser presentados a los usuarios de manera que puedan percibirlos, no pueden ser invisibles a todos sus sentidos.


1.4 Distinguible

Facilitar a los usuarios ver y oír el contenido, incluyendo la separación entre el primer plano y el fondo.


1.4.1 Uso del Color - Nivel A

El color no se usa como único medio visual para transmitir la información, indicar una acción, solicitar una respuesta o distinguir un elemento visual.

Ejemplos

Los campos con etiqueta y bordes en rojo y marcados con un asterisco son obligatorios.

Nombre*:


Apellido:





Otro ejemplo que puede darse en formularios es a la hora de comunicar si un campo ha sido bien introducido o es erróneo.

Usar el formato MM/YYYY.

1.4.1 Uso del Color - Nivel A

Si el audio de una página web suena automáticamente durante más de 3 segundos, se proporciona ya sea un mecanismo para pausar o detener el audio, o un mecanismo para controlar el volumen del sonido que es independiente del nivel de volumen global del sistema.

1.4.3 Contraste (mínimo) - Nivel AA

Todo texto debe mantener un ratio de contraste de mínimo 4.5:1 con el fondo, a excepción de:

Ejemplos

  • Texto grande: el contraste se reduce a 3:1 para textos de mayor tamaño.
  • Casual: para partes inactivas de la interfaz, decoración, partes de una imagen con otro significado o no visibles no se requiere constraste.
  • Logotipos: si el texto es parte de un logo o marca, no hay constraste mínimo.

En este caso, acabamos de usar un color con el mínimo contraste de 4.5:1 con el fondo blanco.

1.4.4 Cambio del tamaño de texto - Nivel AA

A excepción de los subtítulos y las imágenes de texto, todo el texto puede ser ajustado sin ayudas técnicas hasta un 200 por ciento sin que se pierdan el contenido o la funcionalidad.

Ejemplos

En este caso, invitamos a probar libremente con nuestra página:

  • Windows/Linux: Ctrl + (+) o Ctrl + botón central del ratón
  • Mac: Cmd + (+) o Cmd + botón central del ratón
  • En general: opción de zoom en los navegadores

1.4.5 Imágenes de texto - Nivel AA

Si con las tecnologías que se están utilizando se puede conseguir la presentación visual deseada, se utiliza texto para transmitir la información en vez de imágenes de texto, a excepción de estos casos:

Ejemplos

Siempre que sea posible, se presenta la información en texto y se le da formato con las distintas etiquetas para cada propósito.

  • Personalizable: la imagen de texto puede ser personalizada a gusto del usuario.
  • Esencial: una presentación particular del texto es esencial para expresar información.

Otro caso, nosotros simpáticamente queremos hacer referencia a una conocida marca de refrescos. En vez de usar cualquier conversor en la web y tomar una imagen, usamos CSS y presentamos un texto.

Delicioso refresCo azucarado de Cola

1.4.5 Imágenes de texto - Nivel AA

Siempre que la tecnología usada sea capaz de lograr una presentación visual específica, se usará texto para expresar información en lugar de imágenes, a excepción de: imágenes de texto esenciales o personalizables.

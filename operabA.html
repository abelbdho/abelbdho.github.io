2. Operabilidad - Segundo Principio

Todas las funcionalidades serán accesibles mediante el teclado.


2.1 Accesible por teclado

Si toda la funcionalidad está disponible para el uso de teclado, todas las alternativas que lo simulan podrán acceder también (por ejemplo, ingreso por voz).


2.1.1 Teclado - Nivel A

A partir del teclado se podrá acceder a toda funcionalidad del contenido sin restricciones de tiempo, a excepción de funciones que requieren como entrada el trayecto de movimientos del usuario (por ejemplo, pinceladas).

Al pasar el ratón sobre una imagen, esta cambia durante unos segundos. Se adapta para ofrecer una experiencia similar a personas que accedan con el teclado, para el que inicialmente la imagen no captaría el foco.

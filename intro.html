Discapacidad en la web


La accesibilidad es esencial para la creación de páginas web de calidad y para todos. Fundamentalmente, la web esta diseñada para ser usada por cualquier persona, si bien quedan aún muchas barreras por derribar en aras de lograr realmente ese objetivo. La web, que debería permitir el soslayar las limitaciones físicas de cada uno, a veces incluye dificultades añadidas para gente con algún tipo de discapacidad (incluso para gente sin ella). Aproximadamente una quinta parte, un 20%, de la población mundial sufre algún tipo de discapacidad.

Por ello, el objetivo de Web Accesibility Initiative (WAI) de W3C es el de llevar a la web a su máximo potencial permitiendo a todo el mundo el acceso equitativo a la web.




Caso práctico


En la actualidad, Internet se ha convertido en un medio casi indispensable en nuestras vidas, siendo en el caso de personas con discapacidad incluso más necesario. Al afirmar lo antes mencionado se tienen en cuenta todas las discapacidades y a cualquier nivel, así como situaciones o circunstancias en las que nuestros sentidos pueden verse reducidos temporalmente: capacidades visuales, cognitivas, auditivas o motoras.

Para quien lo requiere, el uso de Internet no se reduce a la comunicación o el entretenimiento, puede usarse para realizar la compra diaria online, el correo electrónico, etc; y si bien el formulario de registro en una red social nos puede parece baladí, no ocurriría lo mismo con un formulario para ejercer el derecho a voto por correo o para la solicitud de una tarjeta sanitaria.

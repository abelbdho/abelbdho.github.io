1.4.3 Contraste (Mínimo) - Nivel AA

Todo texto debe mantener un ratio de contraste de mínimo 4.5:1 con el fondo, a excepción de:

  • Texto grande: el contraste se reduce a 3:1 para textos de mayor tamaño.
  • Casual: para partes inactivas de la interfaz, decoración, partes de una imagen con otro significado o no visibles no se requiere constraste.
  • Logotipos: si el texto es parte de un logo o marca, no hay constraste mínimo.

En este caso, acabamos de usar un color con el mínimo contraste de 4.5:1 con el fondo blanco.

1.4.7 Bajo o inexistente ruido de fondo - Nivel AAA

Para contenido auditivo pregrabado que contenga principalmente discurso en primer plano y no sea parte de un CAPTCHA, logo o hilo musical, al menos se cumple una de estas condiciones:

  • Sin ruido de fondo: el audio no contiene ruido de fondo.
  • Apagado: tenga ruido de fondo pero pueda desactivarse.
  • 20 decibelios: a excepción de sonidos de uno o dos segundos, debe haber una diferencia de 20 decibelios entre voz y fondo.


El contenido solo auditivo no debe tener ruido de fondo para facilitar la audición del mensaje principal

En este caso tenemos un mensaje de voz claro y sin ruido de fondo.

1.4.13 Contenido bajo cursor o foco - Nivel AA

Cuando el foco o el puntero se coloca y retira sobre una zona y hace visible contenido escondido, se cumplen lo siguiente:

  • Desechable: excepto en casos de error o en los que no se oculte nada, un mecanismo permitirá volver a esconder este contenido sin mover el puntero o foco.
  • Planeable: si al pasar sobre el contenido se hace visible, se puede interactuar con el sin que desaparezca.
  • Persistente: el contenido permanece hasta movimiento del cursor o foco, que el usuario lo deseche o que no sea útil.

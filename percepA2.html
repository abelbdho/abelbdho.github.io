1. Perceptible - Primer Principio

La información y todo componente presentado en la interfaz de la web deben ser presentados a los usuarios de manera que puedan percibirlos, no pueden ser invisibles a todos sus sentidos.

Para lograr la accesibilidad web hay que ser conscientes de que no todos los usuarios acceden a la Web con los mismos medios técnicos y que no todos los usuarios son capaces de percibir correctamente todos los contenidos publicados en la Web. Esta idea se resume en los dos principios básicos del diseño web accesible:

  • Crea páginas que se transformen correctamente.
  • Ofrece el contenido de manera comprensible para facilitar la navegación por el sitio web.

1.4.3 Contraste (mínimo) - Nivel AA

Todo texto debe mantener un ratio de contraste de mínimo 4.5:1 con el fondo, a excepción de:

Ejemplos

Ejemplos

  • Texto grande: el contraste se reduce a 3:1 para textos de mayor tamaño.
  • Casual: para partes inactivas de la interfaz, decoración, partes de una imagen con otro significado o no visibles no se requiere constraste.
  • Logotipos: si el texto es parte de un logo o marca, no hay constraste mínimo.

En este caso, acabamos de usar un color con el mínimo contraste de 4.5:1 con el fondo blanco.

Ejemplos
